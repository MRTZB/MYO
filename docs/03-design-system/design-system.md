# MYO Design System

## Merkuitstraling
Modern, strak, premium, professioneel en rustig.

## Hoofdkleuren
- Tiffany Blue: #81D8D0
- Hermès Orange: #F37021
- White: #FFFFFF
- Dark: #111827
- Graphite: #1F2937

## Light Mode
- Achtergrond: wit
- Kaarten: wit met zachte schaduw
- Primaire acties: Tiffany Blue
- Accent/waarschuwing: Hermès Orange

## Dark Mode
- Achtergrond: donker antraciet
- Kaarten: graphite
- Primaire acties: Tiffany Blue
- Accent/waarschuwing: Hermès Orange

## UI-regels
- Veel witruimte
- Afgeronde kaarten
- Geen drukke dashboards
- Belangrijkste acties duidelijk zichtbaar
- AI-suggesties prominent maar niet storend
