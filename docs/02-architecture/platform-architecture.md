# MYO Platform Architecture

## Hoofdlagen

MYO bestaat uit:

- Presentation Layer
- Application Layer
- Business Engine
- AI Engine
- Automation Engine
- Integration Layer
- Data Layer

## MYO Core

De core bevat:

- Authentication
- Companies
- Users
- Roles
- Permissions
- Settings
- Modules
- Themes
- Languages
- Notifications
- Audit Logs
- AI
- Automations

## Architectuurregel
Elke nieuwe functie moet modulair, herbruikbaar, beveiligd en tenant-aware zijn.
