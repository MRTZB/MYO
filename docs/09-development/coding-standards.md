# MYO Coding Standards

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- PostgreSQL
- Vercel
- OpenAI API

## Code regels
- TypeScript verplicht
- Geen hardcoded kleuren
- Geen hardcoded teksten
- Alle teksten via language engine
- Alle kleuren via theme tokens
- Componenten herbruikbaar maken
- Business logic niet in UI-componenten plaatsen
- Elke module eigen map
- Security standaard meenemen

## Naamgeving
- Bestanden: kebab-case
- Componenten: PascalCase
- Functies: camelCase
- Database tabellen: snake_case
